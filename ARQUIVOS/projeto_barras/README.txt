Leitor de Códigos de Barras com Descrição

Este script lê códigos de barras e retorna a descrição associada a cada código. Ideal para automatizar a identificação de produtos ou itens a partir do código de barras, facilitando o controle de estoque, inventário ou qualquer aplicação que envolva reconhecimento de produtos.
Funcionalidades

    Leitura de códigos de barras via imagem, webcam ou arquivo.

    Consulta da descrição do produto associada ao código de barras.

    Suporte para diversos tipos de códigos (EAN, UPC, Code128, etc).

    Interface simples para entrada do código e saída da descrição.

Requisitos

    Python 3.x

    Bibliotecas:

        opencv-python

        pyzbar

        requests (caso utilize API externa para consulta)

        Outras conforme necessário

Instalação

pip install opencv-python pyzbar requests
